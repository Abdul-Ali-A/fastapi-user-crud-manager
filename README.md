FastAPI (User CRUD Manager)
Full-stack FastAPI CRUD application with:
Pydantic validation
Credit score auto-delete (<100)
Modern Jinja2 + Tailwind UI (2025 style)

Acknowledgments:
Frontend UI (Jinja2 templates + Tailwind styling) assisted by AI. All backend logic, routes, Pydantic models, and CRUD functionality written 100% by me.

Features:
Create / Read / Update / Delete users
View user by username
Auto-delete low credit score


Tech Stack:
FastAPI
Pydantic
Jinja2 Templates

How To Run Locally
In Terminal:
pip install fastapi uvicorn
uvicorn main_app:app --reload

Open http://127.0.0.1:8000
